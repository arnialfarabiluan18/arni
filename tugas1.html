Perintah Perintah untuk mengupload ke repository github


awalnya kita harus membuat akun pada github.com

git init untuk memulai

git status untuk melihat status

git add untuk menambahkan perubahan yang sudah dibuat pada teks atau projcet kita

git commit -m "pesan commit atau sebuah perubahan"untuk menambahkan pesan untuk mewakili perkerjaan yang telah kita buat sebelumnya

git branch -M untuk memindahkan pesan

git remote add origin https://github.com/arnialfarabiluan18/arni.git untuk menghubungkan dengan project

Ls untuk melihat file dalam repository

git push -u origin main untuk menguplod file ke repository

itulah beberapa tahap dan perintah yang kita pelajari pada pertemuan 2 matakuliah Framework 2
